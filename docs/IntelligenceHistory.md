# Intelligence History

## Purpose

Intelligence History preserves the outputs generated by the Discogs Intelligence Platform over time.

Historical Intelligence analyses how the underlying collection changes between snapshots.

Intelligence History analyses how the platform's conclusions change between intelligence executions.

This distinction allows DIP to answer questions such as:

- Has Collection Health improved or deteriorated?
- Which releases repeatedly appear as Hidden Gems?
- When did a particular insight first appear?
- Has evidence coverage improved?
- Did a module begin failing or skipping executions?
- Did a result change because the module methodology changed?

Intelligence History does not create new intelligence, make collecting decisions or introduce a generic intelligence score.

---

## Core Principle

An Intelligence History record is an immutable observation of what DIP concluded at a particular point in time.

Once recorded, a historical intelligence result must not be edited or overwritten.

Each new engine execution creates a new historical run.
